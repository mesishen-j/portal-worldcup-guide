# portal-worldcup-guide

本仓库用于归档和发布多个独立的 HTML 页面，内容涉及世界杯相关的信息整理与展示。页面均为静态文件，不依赖后端服务，可直接在浏览器中打开查看。

## 项目简介

这是一个以世界杯为主题的页面归档项目。仓库内收录了若干独立制作的 HTML 文件，每个文件均可单独运行，适合作为信息索引、参考手册或轻量级展示页面使用。

## 目录说明

```
portal-worldcup-guide/
├── index.html          # 主入口页面（可选）
├── pages/              # 独立页面文件夹
│   ├── page1.html
│   ├── page2.html
│   └── ...
└── assets/             # 资源文件（样式、脚本、图片等）
    ├── css/
    ├── js/
    └── images/
```

- `index.html`：可选的导航或欢迎页面，方便浏览所有归档页面。
- `pages/`：存放各个独立的 HTML 页面文件，彼此之间无直接依赖。
- `assets/`：存放页面引用的样式表、脚本、图片等公共资源。

## 页面归档说明

- 所有页面均为静态 HTML，无需服务器环境即可运行。
- 每个页面独立维护，修改或新增页面不会影响其他页面。
- 页面内容聚焦于世界杯相关的信息整理，不涉及具体网站或第三方服务。
- 归档页面可能包含表格、列表、链接等常见网页元素，但所有链接均指向本地或公开资源。

## 维护说明

- 欢迎提交 Issue 或 Pull Request 来完善页面内容、修复错误或优化样式。
- 新增页面时，请将 HTML 文件放入 `pages/` 目录，并确保资源引用路径正确。
- 若添加新的资源文件（如 CSS、JS、图片），请放入 `assets/` 对应子目录。
- 提交前请检查页面在主流浏览器（Chrome、Firefox、Edge 等）中的显示效果。
- 本仓库不包含任何商业推广或诱导点击内容，请保持内容的纯粹与客观。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接在浏览器中打开 `index.html` 或 `pages/` 下的任意 HTML 文件。
3. 也可通过 GitHub Pages 功能在线访问（若已启用）。
